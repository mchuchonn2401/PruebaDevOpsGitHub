## Descripción

Esta es una calculadora básica que permite realizar operaciones matemáticas fundamentales:
- Suma (+)
- Resta (-)
- Multiplicación (×)
- División (÷)

## Archivos del proyecto

```
├── index.html      # Estructura HTML de la calculadora
├── style.css       # Estilos y diseño visual
├── script.js       # Lógica y funcionalidad JavaScript
├── README.md       # Documentación del proyecto
└── .gitignore      # Archivos a ignorar por Git
```

## Cómo usar

1. **Abrir el proyecto**: Simplemente abre el archivo `index.html` en tu navegador web
2. **Usar con mouse**: Haz clic en los botones para introducir números y operaciones
3. **Usar con teclado**: 
   - Números: 0-9
   - Operaciones: +, -, *, /
   - Calcular: Enter o =
   - Borrar: Backspace
   - Limpiar: Escape o C

## Tecnologías utilizadas

- **HTML5**: Estructura del documento
- **CSS3**: Estilos y diseño visual
- **JavaScript**: Lógica de la calculadora
